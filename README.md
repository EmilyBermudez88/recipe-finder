# Recipe Finder

A web application that helps users discover recipes based on ingredients, cuisine, and preparation time. Built with React and TypeScript, this app integrates with athe spoonacular API to provide real-time search results and detailed cooking information.

## Tech Stack

Frontend: React 18+
Language: TypeScript
State Management: Context API
Styling: SASS / CSS3
API: Spoonacular API: https://spoonacular.com/food-api

## Installation & Setup

```
git clone https://github.com/EmilyBermudez88/recipe-finder.git
npm install
npm run dev
```
Create a .env file in teh root directory and add your API key

```
npm run dev
```

Then enjoy!
